# Portfolio CMS Backend

Production-ready portfolio backend built with:

* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* Helmet Security
* Rate Limiting
* Cloudinary Integration (In Progress)

## Features

* Authentication
* Projects Management
* Skills Management
* Certificates Management
* Contact Form
* Resume Management
* Dashboard Statistics

## Run Locally

npm install

npm run dev

## Environment Variables

Create .env file using .env.example
